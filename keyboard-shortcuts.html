Keyboard Shortcuts

Keypress Command Result
Sublime Text
CMD + → Command + Right Arrow Move to last character of line
CMD + / Command + forward slash Toggle line comments
Atom
CMD + → Command + Right Arrow Move cursor to the end of the line
CMD + ← Command + Left Arrow Move cursor to the beginning of the line
⌥ + → Option + Right Arrow Move cursor to the end of the next word
⌥ + ← Option + Left Arrow Move cursor to the beginning of the previous word
‸ + M Control + M Go to matching bracket
CMD + K + ↓ Command + K + down arrow Split panes vertically
CMD + K + → Command + K + right arrow Split panes horizontally
CMD + / Command + forward slash Toggle line comments
CMD + D Command + D Multi-select
Google Chrome Shortcuts
CMD + R Command + R Refresh Page (to see new edits)
⌥ + CMD + I Option + Command + I Open your Chrome Developer Tools
Mac Based Shortcuts
CMD + Tab Command + Tab Switch between open applications
CMD + Command + plus Increase font size
CMD - Command + minus Decrease font size
CMD + ⇧ + 4 Command + Shift + 4 Take a screenshot (saves to desktop)
